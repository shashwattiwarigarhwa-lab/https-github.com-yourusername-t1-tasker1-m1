# Project UI Notes

This folder contains project-level screens and forms:

- `project-list.tsx`: user's accessible projects
- `create-project-form.tsx`: Admin bootstrap flow because creator becomes Admin
- `project-detail.tsx`: project dashboard, member panel, and task board
- `member-manager.tsx`: add/remove project members

The UI is intentionally direct and interview-friendly: no hidden workflow, clear CRUD actions, and visible project membership.
